MangaDex API Documentation

A comprehensive API for manga readers. Uses a single project account for authenticated requests.

Tip:
  • To find a Manga ID, use the /manga search endpoint.
  • To find a Chapter ID or Group ID, use the /manga/:id/feed endpoint for that manga.

Manga & Chapters
